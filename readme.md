# Bring home test

**Frontend: http://localhost:3000/**

## Running frontend

cd into folder: client

```
cd client
```

install dependencies

```
npm install
```

start

```
npm run start
```

view website

```
http://localhost:3000/
```

## Running backend

cd into folder: server

```
cd server
```

install dependencies

```
npm install
```

start

```

npm run server

```
